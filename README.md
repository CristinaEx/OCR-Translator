# a translator which based on screen

- 基于屏幕取词的翻译软件

## 注意！

- 本项目是半成品，因为tesseract的识别率较低，所以放弃该项目。

- 预览，清空和更多未完成。

- 内置百度OCR，tesseract，谷歌翻译，百度翻译的接口（两家百度收费，需要开发者账号）。
